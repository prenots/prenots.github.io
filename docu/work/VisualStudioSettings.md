# Visual Studio Settings

The following settings are available for Visual Studio 2026.

## Brace Colorization

> Tools > Options > Text Editor > General > Display

:far fa-square-check fa-fw:
Enable brace pair colorization

## Tabs in multiple Rows

> Tools > Options > Environment > Tabs > Document Tabs

:far fa-square-check fa-fw:
Show tabs in multiple rows

## New Tabs to the right

> Tools > Options > Environment > Tabs > Document Tabs

:far fa-square-check fa-fw:
Insert new tabs to the right of existing tabs

## Colorize Tabs by Project

> Tools > Options > Environment > Tabs > Document Tabs

:far fa-square-caret-down fa-fw:
Tab colorization method: Project

## Pinned Tabs in separate row

> Tools > Options > Environment > Tabs > Document Tabs

:far fa-square-check fa-fw:
Show pinned tabs in a separate row

## View White Spaces

> Edit > Advanced

:fas fa-toggle-on fa-fw:
View White Space

## Ligature (single symbol for == and !=)

> Tools > Options > Environment > Fonts and Colors

:far fa-square-caret-down fa-fw:
Font: Cascadia Code

## Sync Active File

> Tools > Options > Projects and Solutions > General

:far fa-square-check fa-fw:
Track Active Item in Solution Explorer

## Subword Navigation

> Tools > Options > Environment > Keyboard

:far fa-keyboard fa-fw:
`Edit.NextSubword` (Text Editor) CTRL + Right Arrow

:far fa-keyboard fa-fw:
`Edit.NextSubwordExtend` (Text Editor) CTRL + SHIFT + Right Arrow

:far fa-keyboard fa-fw:
`Edit.PreviousSubword` (Text Editor) CTRL + Left Arrow

:far fa-keyboard fa-fw:
`Edit.PreviousSubwordExtend` (Text Editor) CTRL + SHIFT + Left Arrow
